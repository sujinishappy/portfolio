04. Seoul Jazz Festival

Description

서울 재즈 페스티벌을 홍보하기 위한 인터랙티브 웹 사이트를 기획하였습니다.

자바스크립트의 옵저브 기능을 이용해 해당하는 컨텐츠가 보이기 시작하면,

인터랙티브 효과들이 호출 되어 동적으로 제작하였습니다.

음악과 관련된 주제이기 때문에 직선보다는 곡선의 원이 잘 어울릴 것 같아,

원 오브젝트를 웹 사이트에 추가적으로 많이 활용해보았습니다.

효율적인 코드 작성과, 스크립트의 두려움을 없애 주었던 프로젝트입니다.
